Arrays

Introduction to Arrays

An array is a variable which can hold more than one value at a time.

The syntax of an array is: array_name = [item1, item2, item3, ...];. For example:

var cars = ["Saab", "Volvo", "BMW"];

The keyword 'new' can also be used to create an array and assign values. For example:

var cars = new Array("Saab", "Volvo", "BMW");

They both do the same, so stick to the first where possible as the second is outdated.

Array Elements

Accessing Array Elements

You can access the full array of data using just the array name.

You can also access a specific array element using the index number. Remember, in JS zero is the first number. For example:

var name = cars[0]; // Accesses Saab, the first element in the cars array.

Adding Array Elements

To add a new element to the end of an array, use the push() method. For example:

fruits.push("Lemon"); // Adds "Lemon" as the last element.

Additionally, a new element can be added to the end using the length property. For example:

fruits[fruits.length] = "Lemon"; // Adds "Lemon" as the last element.

Or directly with an index number:

fruits[6] = "Lemon"; // Adds "Lemon" as the 6th element.

...although this should be used with caution, as it could result in undefined 'holes' in the array.

Changing Array Elements

You can also use the array index to update an array element. For example:

cars[0] = "Ford"; // Updates the first element from Saab to Ford.

Arrays & Objects

Arrays Are Objects

Arrays are special types of objects. The typeof operator in JS returns "object" for arrays. However, it's best to refer to arrays as arrays, rather than objects, as:

  • Arrays use index number to access it's "elements", as below.
  • Objects use names to access it's "members", as below.

var person = ["Dan", "B"]; // An Array.
person[0]; // Returns "Dan".

var person = {firstName:"Dan", lastName:"B"}; // An Object.
person.firstName // Returns "Dan".

Remember that array elements don't just have to be string or numbers; they can also be objects too. For example, an array element could be a function.

Associative Arrays

Many programming languages support associative arrays; where associative arrays are those with named indexs. However, JS does not support arrays with named indexes. JS always uses numbered index. Note that associative arrays are also called hashes.

Naming an index turns the array into an object, which then changes the behaviour of the 'array'.

The Difference Between Arrays & Objects

In JavaScript:

  • Arrays used numbered indexes;
  • Objects used numbered indexes.

...where an array is just a special type of object.

So, use objects when element names should be string; user arrays when the element names can be numbers.

Array Properties & Methods

Arrays have a variety of properties and methods which can act on array elements.

The length Property

The lenght propery of an array returns the length of the array, i.e. how many elements it contains. For example:

var fruits = ["Banana", "Orange", "Apple", "Mango"];
fruits.length; // Returns 4.

A useful thing to remember is that the lenght of an array is always one higher than the index ID of the final array element. You can use this to capture the last array element. For example:

var first = fruits[0]; // Returns the first element, Banana.
var fruit = fruits[fruits.length - 1]; // Returns the last element, Mango.

The toString() & join() Methods

The toString() method converts an array to a string of comma separated values. For example:

var fruits = ["Banana", "Orange", "Apple", "Mango"];
fruits.toString(); // Returns "Banana,Orange,Apple,Mango"

Note that instinctively, in JS "fruits" will also return comma separated string in the same way that "fruits.toString()" does.

The join() method also converts to string, but unlike toString(), you can specify the separator when using join(). For example:

fruits.join(" * "); // Returns "Banana * Orange * Apple * Mango"

The pop() & push() Methods

Popping and pushing is the addition and removing of elements. You pop items OUT of an array, and push items INTO an array.

The pop() methods removes the last element from an array. For example:

var fruits = ["Banana", "Orange", "Apple", "Mango"];
fruits.pop(); // Removes the last element "Mango".

The pop() method returns the value that was popped out, so can be assigned to a variable. For example:

var x = fruits.pop(); // Assigns "Mango" to x.

The push() method adds a new element to the end of an array. For example:

fruits.push("Kiwi"); // Adds new element "Kiwi" to the end of the array.

The push() method returns the new length of the array when the new element is added. This new length can be assigned to a variable. For example:

var x = fruits.push(); // Assigns 5 to x; the new array length.

Shifting Elements

Shifting is the same as popping, but this time working on the first element instead of the last element. The shift() method removes the first element in the array, and 'shifts' all other elements to a lower index. For example:

var fruits = ["Banana", "Orange", "Apple", "Mango"];
fruits.shift(); // Removes the first element "Banana" from the list and shits others down.

The shift() method returns the element shifted out, so this can be assigned to a variable. For example:

var x = fruits.shift(); // Assigns "Banana" to x.

Unshifting is the same as pushing, but this time adding the new element as the first element rather than the last. The unshift() method adds the new element as the first index, and shifts the existing elements to a higher index.

fruits.unshift("Lemon"); // Adds "Lemon" as the first element and shifts others up.

The unshift() method returns the new length of the array when the new element is added. This new length can be assigned to a variable. For example:

var x = fruits.unshift("Lemon"); // Assigns 5 to x; the new array length.

Merging (Concatenating) Arrays

The concat() method creates a new array by merging existing arrays. For example:

var myGirls = ["Amy", "Katie"];
var myBoys = ["Ashley", "Liam"];
var myChildren = myGirls.concat(myBoys); // Joins myBoys to myGirls.

The concat() method doesn't change the existing arrays; it creates a new one.

The concat() method can take multiple arguments. For example:

var array1 = ["A", "B", "C"];
var array2 = ["D", "E"];
var array3 = ["F", "G"];
var newArray = array1.concat(array1, array2);

The concat() method can also take values as arguments. For example:

var array1 = ["A", "B", "C"];
var newArray = array1.concat(["D", "E", "F", "G"]);

Splicing An Array

Splicing an array is allows new elements to be added into any position within an array. The splice() method has a few parameters to define. For example:

var fruits = ["Banana", "Orange", "Apple", "Mango"];
fruits.spluce(2, 0, "Lemon", "Kiwi");

In the example above, the first parameter "2" defines the index position where the new elements should be added. The second parameter "0" defines how many existing elements should be removed. The rest of the parameters defined the new elements to be added.

The splice() method can be used to remove elements from any point within an array using the paramters. For example:

fruits.splice(1,2); // At index position 1, 2 elements are removed.

...this prevents the array from having undefined holes. Those removals could be replaced too. For example:

var fruits = ["Banana", "Orange", "Apple", "Mango"];
fruits.splice(1,2,"Lemon","Kiwi") // Array becomes "Banana", "Lemon", "Kiwi", "Mango"

Note that splice returns the deleted elements; these can be assigned to a variable.

Slicing An Array

Slicing an array takes a piece out of an array and puts it into a new array using the slice() method. For example:

var fruits = ["Banana", "Orange", "Apple", "Mango"];
var newFruits = fruits.slice(3); // slices out index 3 ("Apple") onwards and assigns to new array.

Typically, slice will take two arguments, saying the starting index up to (but not including) the end argument. For example:

var fruits = ["Banana", "Orange", "Apple", "Mango"];
var citrus = fruits.slice(1,3)

...this takes index 1 "Orange" to index 2 "Lemon", as the second argument is 'up to but not including. [Remember, in JS indexes start at zero.]

When slicing an array, the original array remains unchanged. The new array contains the sliced elements.

Identifying Arrays

In JS, you can't just use the typeof operator to see if something is an array, as an array is technically an object. For example:

var fruits = ["Banana", "Orange", "Apple", "Mango"];

typeof fruits; // returns object

You can use a new method Array.isArray(). For example:

Array.isArray(fruits); // returns true

Or you can yse the instanceof operator, for example:

fruits instanceof Array; // returns true

Deleting Elements

Elements can be deleted from an array using the delete operator. For example:

var fruits = ["Banana", "Orange", "Apple", "Mango"];
delete fruits[0]; // Changes the first element from "Banana" to undefined.

...although it's best to avoid using this as it leaves 'holes' in the array. Use pop() or shift() instead, using splice() if required.
