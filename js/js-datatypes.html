Introduction to Data Types

In JavaScript, there are 7 basic data types:

  1. Number
  2. String
  3. Boolean
  4. "null"
  5. "undefined"
  6. Objects
  7. Symbols

The 'typeof' operator can be used to identify which type of data is stored in a variable. For example:

typeof(x); // or
typeof x;

//returns the data value, e.g."string"
//note that data type 'null' returns 'object' which is an error in the langauge.
