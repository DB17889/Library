Logical Operators Introduction

There are three logical operators in JS:

  1. || (OR)
  2. && (AND)
  3. ! (NOT)

Although they are called logical operators, they can be applied to values of any type and not just boolean values.

|| (OR)

The "OR" operator is represented with: ||

Traditionally in programming, logical OR is meant for boolean values only. For example:

alert(true || true); // true
alert(true || false); // true
alert(false || true); // true
alert(false || false); // false

If the operand is NOT a boolean, it is converted to one for evaluation. For example:

if (1 || 0) {
___alert('true');
}
// works like if (true or false) alert ('true').

As a more relevant example:

let hour = 9;

if (hour < 10 || hour > 18) {
___alert('The office is closed');
}

So essentially, if < 10 is true or hour > 18 is true, then the message is shown.

More conditions can be set. For example:

let hour = 12;
let isWeekend = true;

if (hour < 10 // hour > 18 || isWeekend) {
___alert('The office is closed');
}

...so if any of those arguments are true, the message is returned.

OR Finds The First Truthy Value

Using the following example with multiple OR values:

result = value1 || value2 || value3;

The OR || operator here is doing the following:

  • Evaluates from left to right.
  • Converts each operand to boolean. If the result is true, the process stops and returns the original value of that operand. If false, it moves on to the next operand.
  • If all operands are evaluated and are boolean false, the last operand is returned.

The value that is returned is the original value, not the boolean counterpart. For example:

alert(1 || 0); // returns '1' as 1 is truthy
alert(true || 'no matter what'); // returns 'true'
alert(null || 1); // returns '1' as it is the first truthy value
alert(null || 0 || 1); // returns '1' as it is the first truthy value
alert(null || undefined || 0); // returns '0' as all are falsy, and 0 is the last value

OR || can be used for a variety of things...

Getting the first truthy value from a list of variables or expressions.

As an example, say there are a list of variables that could contain data, be null or be undefined. How do you get the first one with date? Use OR || like this:

let currentUser = null;
let defaultUser - "Dan";

let name = currentUser || defaultUser || "unnamed";

alert(name); // returns 'Dan' as defaultUser is the first truthy value.

In the example above, if there is no currentUser or defaultUser, then 'unnamed' is returned.

Short-circuit evaluation

Say for example you only need to assign a valiable if another value is false. As OR || evaluates from left to right, it stops evaluationg after the first truthy value is found. For example:

let x;

true || (x = 1);
alert(x); // undefined, as x=1 is not evaluated as true is truthy.

null || (x = 1);
alert(x); returns 1, as null is bypassed and x=1 is truthy so is evaluated and run.

...in general though, it's safer to use a regular 'if', covered later.

&& (AND)

The "AND" operator is represented with: &&

Traditionally in programming, logical AND is meant for boolean values only. If both operands are truthy, true is returned. Otherwise, false is returned. For example:

alert(true && true); // true
alert(true && false); // false
alert(false && true); // false
alert(false && false); // false

As an example with 'if':

let hour = 12;
let minute = 30;

if (hour == 12 && minute == 30) {
___alert('The time is 12:30');
}

As with OR, any value is allowed as an AND operand:

if (1 && 0) {
___alert("wont work as result is falsy");
}

AND Finds The First Falsy Value

Where OR || finds the truthy value, AND && finds the first falsy value

Using the following example with multiple AND values:

result = value1 && value2 && value3;

The AND && operator here is doing the following:

  • Evaluates from left to right.
  • Converts each operand to boolean. If the result is false, the process stops and returns the original value of that operand. If true, it moves on to the next operand.
  • If all operands are evaluated and are boolean true, the last operand is returned.

For example:

alert(1 && 0); // returns 0
alert(1 && 5); // returns 5
alert(null && 5); // returns null
alert(0 && 'no matter what'); // returns 0

alert(1 && 2 && null && 3); returns null
alert(1 && 2 && 3); returns 3

THe precendence of AND and OR is that AND is greather than OR, so && is greater than ||. For example:

a && b || c && d // && takes precendence over ||, so its essentially:
(a && b) || (c && d);

! (NOT)

The boolean NOT operator is represented with !. For example:

result = !value;

The operator accepts a single argument and:

  • Converts the operand to boolean true/false.
  • Returns the inverse value.

For example:

alert(!true); // returns false
alert(!0); // returns true

A double NOT !! canbe used to conver a value to boolean type.

The precedence of NOT has priority over AND and OR, so the order of precedence is: !, &&, ||
