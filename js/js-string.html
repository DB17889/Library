Introduction to Strings

String can be contained within either single quotes or double quotes. Ideally, stick to the same one throughout code. For example:

let a = "text" // this is fine
let b = 'text' // this is fine
let c = "text' // this will return error
let d = text // this will return error

If single or double quotes are required within a string, the \ escape character may be required. For example:

let a = "This is a "quote" type quote" // this will work
let b = 'I'm typing a quote' // this will not work

To prevent the single quote in "I'm" from closing the string quote, us \ to escape it, like:

let b = 'I/'m typing a quote' // this will work

Concatenating Strings

String can be concatenated using the + operator. For example

let one = "Hello, ";
let two = "how are you?";
let joined = one + two;
joined; //returns: Hello, how are you?

A string can be concatenated multiple times. For example:

let multiple = one + one + one;
multiple; // returns: Hello, Hello, Hello,

And both variables and string can be concatenated, like this:

let lyric = one + "is it me you/'re looking for?";
lyric; // returns: Hello, is it me you're looking for?

String & Numbers

If you concatenate a number and a string, JS cleverly converts the number into string and joins them. For example:

'Word' + 123; // 123 converted to string and returns: Word123

You can even do this with just numbers, by forcing the number into string by using quote marks to wrap it in. For example:

let date = "19" + "89"; date; returns: 1989

To convert a number/string data type, use Number() or toString():

let myString = "123";
let myNum = Number(myString);
typeof myNum; // returns: number

let myNum = 123;
let myString = nyNum.toString();
typeof myString; // returns: string

Note the difference syntax, i.e. 'Number(myString)' vs 'myNum.toString().

In real life, this can be useful when converting numeric values entered into a text form by a user.

var txt = "ABCDE";
var txtlen = txt.length; // Returns '5'.

Find String Within a String

Use the indexOf() method to return the position (or 'index') of the FIRST occurance of the specified string. For example:

var str = "Please locate where the word locate occurs.";
var pos = str.indexOf("locate"); // Returns '7' to indicate the 7th character where it starts.

Use the lastIndexOf() method to return the index of the LAST occurance of the specified string. For example:

var str = "Please locate where the word locate occurs.";
var pos = str.indexOf("locate"); // Returns '21' to indicate the 21sa character where it starts

For both indexOf() and lastIndexOf(), if no string is found, '-1' is returned.

Both indexOf() and lastIndexOf() also accept a second parameter. This determines the starting position for the search. For example:

var str = "Please locate where the word locate occurs.";
var pos = str.indexOf("locate", 15); // Returns '21', as the STARTING position of the search was character 15.

...where lastIndexOf() works backwards. For example:

var str = "Please locate where the word locate occurs.";
var pos = str.lastIndexOf("locate", 15); // Returns '7', as the STARTING position of the search was character 15, working BACKWARDS.

Note that the search() method does the same as the indexOf() method, however there are key differences between the two:

  • The search() method cannot take a start position arguement.
  • The indexOf() method cannot take powerful search values (regular expressions).

Extracting String Parts

There are three methods of extracting part of a string:

  1. slice(start, end)
  2. substring(start, end)
  3. substr(start, lenght)

The slice() Method

The slice() method extracts part of a string and returns the extracted part as a new string. The method takes two paratmeters: (1) The start position; (2) The end position (exclusive). For example:

var str = "Apple, Banana, Kiwi";
var res = str.slice(7, 13); // Returns "Banana", i.e. positions 7 to 12 (as 13 is excluded).

REMEMBER: JS counts start at zero, so the first position is 0.

You can als work backwards from the end of a string. For example:

var str = "Apple, Banana, Kiwi";
var res = str.slice(-12, -6); // Returns "Banana", i.e. 12th position from the end to the 6th position from the end (exclusive).

If the second parameter is omitted, the method will slice the rest of the string out. For example:

var str = "Apple, Banana, Kiwi";
var res = str.slice(7); // Returns "Banana, Kiwi" var rez = str.slice(=12); // Returns "Banana, Kiwi"

The substring() Method

The substring() method is similar is slice(), although substring() is unable to accept negative index arguements.

The substr() Method

The substr() method is similar to slice(), the key difference being that the second parameter specifies the lenght of the extracted string, not the end point. For example:

var str = "Apple, Banana, Kiwi";
var res = str.substr(7, 6); // Returns "Banana", i.e. starts at position 7 and picks out 6 characters.

As with slice(), you can use a negative value as the first argument to pick the starting point from the end of the string. You can also omit the second argument to pick out the rest of the string.
