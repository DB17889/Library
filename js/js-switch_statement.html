Introduction To The 'switch' Statement

The if...else statements are good for enabling conditional code, but are usually only good for a small number of choices, or where conditions are complex (e.g. with multiple logical operators). For cases where a selected choices just needs to print out a statement depending on the condition, the syntax can be a little 'over the top'.

This is where 'switch' statements comes in useful. They take a simple expression (or value) as an input, and look though a list of choices before one is found. The relevant code then gets executed. As an example, here is the basic syntax:

switch (expression) {
___case choice1:
______run this code
______break;

___case choice2:
______run this code instead
______break;

___// and so on, for as many choices as required.

___default:
______actually, just run this code
}

In summary, the syntax is this:

  • The 'switch' keyword followed by parenthesis.
  • The epxression or value is contained within the paranthesis.
  • The 'case' keyword followed by a choice that the expression/value could be, followed by a colon.
  • Code that should run if the choice matches the expression is then shown.
  • Next, a 'break' statement followed by a semi colon. If the choice matches the expression/value, then the browser stops executing the code block here.
  • Further choices for evaluation until the break statement is hit.
  • the 'default' keyword is included, followed by code which should run if none of the choices match.
