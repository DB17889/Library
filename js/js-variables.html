Introduction to Variables

A variable is essentially a named storage area for data. To create a variable, use the 'let' keyword. (You may see 'var' instead of 'let'; this is fine, but try and stick to 'let' as this is newer.) For example, to create a variable, or 'define' / 'declare' a variable:

let message; // declares a variable called 'message'

...this is just an empty variable. To assign data to the variable, use the assignment operator as below:

let message;
message = "Hello World!";

...this then stores the string data 'Hello World!' to the memory area associated with the variable. This can be viewed using the alart command:

alert(message); // returns 'Hello World!'

For simplicity, the variable declration and the assignment of data can be completed in one go:

let message = 'Hello World!'; // declares the variable and assigns data in one go

You can also declare multiple variables and assign data to them in one go too, for example:

let user = 'Dan', age = 29, message = 'Hello World!';

...HOWEVER, whilst this might seem shorted, its not recommended from a code readability perspective. Declaring across multiple lines is the 'correct' thing to do, for example:

let user = 'Dan';
let age = 29;
let message = 'Hello World!';

Or, you may see this as:

let user = 'Dan'
, age = 25
, message = 'Hello World!';

A variable can be changed later in the code. For example:

let message = 'Hello!'; // value stored as 'Hello!'
message = 'Goodbye!'; // value changed to 'Goodbye!'

And a variable value can be assigned to another variable, using the variable. For example;

let hello = 'Hello!';
let message;

message = hello;

alert(hello); // returns 'Hello!'
alert(message); // returns 'Hello!'

Naming Variables

There are a couple of restrictions when it comes to naming Variables:

    The name can only contain letters, digits or the symbols $ and _.
    The first chracter must not be a character.

Often is JS, variables are named using camelCase if the variable is more than one word long, where the first words first character is lower case. For example: variableName or thisIsMyVariable.

Note that case DOES matter. For example, a variable called 'variable' is not the same as a variable called 'vARIABLE'.

There are a list of reserved words that cannot be used as variables, as JS uses them for specific functions, for example 'let', 'class' and 'function'. A full list of reserved words is available here.

Constant Variables

To declare a constant variable, i.e. one that does not change, use 'const' instead of 'let'.

const myBirthday = '17.08.1989';

Variables declared using const are called 'constants'. Any attempt to change the data will result in error. For example:

const myBirthday = '17.08.1989';
myBirthday = '01.01.1989'; // results in error as cannot reassign a constant.

Uppercase Constants

It's common practise for constants to be used as a shortcut for commonly used difficult-to-remember values known before execution. This common practise typically uses capital letters and underscores. For example:

const COLOR_RED = "#F00";
const COLOR_GREEN = "#0F0";
const COLOR_BLUE = "00F";
const COLOR_ORANGE = "#FF7F00"

//meaning later, to pick a color use:
let color = COLOR_ORANGE;
