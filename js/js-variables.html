Introduction to Variables

A variable is essentially a named storage area for data. To create a variable, use the 'let' keyword. (You may see 'var' instead of 'let'; this is fine, but try and stick to 'let' as this is newer.) For example, to create a variable, or 'define' / 'declare' a variable:

let message; // declares a variable called 'message'

...this is just an empty variable. To assign data to the variable, use the assignment operator as below:

let message;
message = "Hello World!";

...this then stores the string data 'Hello World!' to the memory area associated with the variable. This can be viewed using the alart command:

alert(message); // returns 'Hello World!'

For simplicity, the variable declration and the assignment of data can be completed in one go:

let message = 'Hello World!'; // declares the variable and assigns data in one go

You can also declare multiple variables and assign data to them in one go too, for example:

let user = 'Dan', age = 29, message = 'Hello World!';

...HOWEVER, whilst this might seem shorted, its not recommended from a code readability perspective. Declaring across multiple lines is the 'correct' thing to do, for example:

let user = 'Dan';
let age = 29;
let message = 'Hello World!';

Or, you may see this as:

let user = 'Dan'
, age = 25
, message = 'Hello World!';

A variable can be changed later in the code. For example:

let message = 'Hello!'; // value stored as 'Hello!'
message = 'Goodbye!'; // value changed to 'Goodbye!'

And a variable value can be assigned to another variable, using the variable. For example;

let hello = 'Hello!';
let message;

message = hello;

alert(hello); // returns 'Hello!'
alert(message); // returns 'Hello!'
